# 퀘스트핏 (QuestFit)
#### 퀘스트핏 (QuestFit) – 스마트 헬스 헬퍼 서비스

<br>

**퀘스트핏**은 업적과 퀘스트를 통한 보상 시스템을 통해 사용자의 운동에 대한 동기부여를 강화합니다. 
<br>
<br>
사용자는 목표를 설정하고, 이를 달성하는 과정에서 얻은 경험치나 보상을 통해 자신의 진전을 시각적으로 확인할 수 있습니다.
<br>
<br>
AI는 카메라를 통해 사용자의 운동 방식을 분석하여 올바른 운동 방법을 유도합니다.
